# REQUIREMENTS

You need a C++ compiler, gstreamer-0.10 or gstreamer-1.0, gtk2.0 and ncurses to
build gst123.

# INSTALLATION

In short, gst123 needs to be built and installed with:

	./configure
	make
	make install
